# 🔍 AI Bottleneck Diagnostic

A web app that analyzes AI/ML code and instantly identifies performance bottlenecks with suggested fixes.

## 🚀 Live Demo
https://ai-bottleneck-diagnostic-3m371g6mr-vasavi1.vercel.app

## 🛠️ Built With
- React + Vite
- Tailwind CSS
- Static code analysis engine

## ✨ Features
- Paste any PyTorch / TensorFlow code
- Instantly detects bottlenecks (Data, Compute, Memory, Inference, Pipeline)
- Shows severity (Critical / Warning / OK)
- Suggests fixes for each bottleneck

## 🏃 Run Locally
npm install
npm run dev
